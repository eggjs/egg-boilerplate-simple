## Installation

**Don't `npm i` this directly**

```bash
$ npm i -g egg-init
$ egg-init --type simple showcase
$ cd showcase
$ npm i
$ npm run dev
$ open http://localhost:7001
```

See [egg-init](https://github.com/eggjs/egg-init) for more detail.
